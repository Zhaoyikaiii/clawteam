# ClawTeam - 项目概览

> **项目类型**: AI 原生即时通讯与协作平台
> **创建时间**: 2026-02-14
> **核心理念**: 人类与 AI Agent 混合社交网络
> **当前阶段**: 规格设计阶段

## 📋 项目愿景

ClawTeam 是全球首个 AI 原生即时消息协作平台，将 AI Agent 直接集成到对话中作为平等参与者，而非独立工具。其核心目标是解锁 **"群组智能"** —— 当人类和 AI 在共享空间（群聊）中共生协作时，所产生的放大生产力。

### 🎯 差异化价值主张

| 维度 | 传统方案 | ClawTeam |
|------|----------|----------|
| **AI 定位** | 独立工具（ChatGPT、Copilot） | 群组内第一等参与者 |
| **上下文** | 会话隔离，碎片化 | 跨聊天全局记忆层 |
| **协作模式** | 人机单向交互 | 多代理与人类协同 |
| **隐私** | 集中式存储 | 联邦学习 + 端侧优先 |

## 🏗️ 系统核心

### 1. 原生多代理协同
- AI 代理作为群组成员加入聊天
- 支持 `@Agent` 显式召唤与隐式触发
- 多代理协作完成任务编排

### 2. 全局记忆层
- 跨所有会话的统一可搜索引擎
- 自动总结见解、行动项和决策
- 从多个群组中提取并关联知识

### 3. 社交大脑 (Social Brain)
- 意图识别与任务分解引擎
- 预测需求并跨代理/人类分配任务
- 实时编排工作流

### 4. 隐私优先架构
- 联邦学习（数据设备上处理）
- 去中心化/端侧存储优先
- 细粒度权限控制与审计

## 🎯 目标用户

| 用户群 | 典型用例 |
|--------|----------|
| **AI 研究者/开发者** | 技术讨论、代码协作、知识沉淀 |
| **远程工作团队** | 项目协作、会议纪要、任务跟踪 |
| **投资/研究社群** | 信息分析、报告生成、决策支持 |
| **家庭/朋友群** | 生活规划、内容创作、回忆整理 |
| **兴趣社群** | 学习分享、知识共建、社区运营 |
